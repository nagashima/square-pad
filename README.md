# Square

画像の長辺を一辺とする正方形にし、短辺方向を白でパディングする PWA。
Instagram への正方形投稿用に、Android 単体でワークフローを完結させるために作成。

## 特徴

- **完全自動**: 画像を渡すだけで長辺基準の正方形化（手動ズーム・トリミング不要）
- **端末内処理**: 画像はサーバーに送信されない（Canvas API でローカル完結）
- **広告・透かしなし**
- **オフライン動作**: PWA としてインストール後はネットワーク不要
- **保存・共有の両対応**: ローカルダウンロードと共有シート経由の送信を選択可能

## 使い方

1. ブラウザでアクセス
2. （任意）ホーム画面に追加して PWA としてインストール
3. 画像を選択
4. 必要に応じて「1440 にリサイズ」のチェックを外す（デフォルト ON）
5. 「保存」または「共有」をタップ

## アルゴリズム

```
入力: 任意縦横比の画像
1. side = max(width, height)
2. side × side の canvas を白で塗りつぶし
3. 元画像を中央配置
4. 1440 リサイズが ON かつ side > 1440 なら 1440 にダウンスケール
5. JPEG (品質 0.95) で出力
```

## 技術スタック

- Vanilla JavaScript（フレームワーク・ビルド不要）
- Canvas API（画像処理）
- Service Worker（オフライン対応）
- Web Share API（共有送信）

## 開発

依存なし。ローカルで動かす場合は同梱の起動スクリプトを使用:

```sh
./serve.sh           # http://localhost:8000/
./serve.sh 8080      # ポート指定
```

中身は `python3 -m http.server` のラッパー。任意の静的サーバーで `index.html` を配信できれば代替可能。

PWA 機能（Service Worker / インストール）は HTTPS または localhost が必要。
